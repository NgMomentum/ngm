A progress bar directive that is focused on providing feedback on the progress of a workflow or action.

It supports multiple (stacked) bars into the same <progress>element or a single <progressbar>elemtnt with optional maxattribute and transition animations.

Settings

<progressbar>

  • value : The current value of progress completed.

  • type (Default: null): Style type. Possible values are 'success', 'warning' etc.

  • max (Default: 100): A number that specifies the total value of bars that is required.

  • animate (Default: true): Whether bars use transitions to achieve the width change.

Stacked

Place multiple <bars>into the same <progress>element to stack them. <progress>supports maxand animate& <bar>supports valueand typeattributes.
