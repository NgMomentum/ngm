ui-css

Allows you to choose the position of the linked CSS files in the HTML's head

This directive is optional. If it's not applied, newly added stylesheets will simply be appended as the last node in the head node. If you for example have stylesheets that need to be loaded after all the Angular UI Router CSS stylesheets, this directive will help solve this problem.

Place the following line somewhere in your head like this: 

<html>
<head>
	<link ui-css>
</head>
</html>

Note: it can only be the head of your document
