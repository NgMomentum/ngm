AngularJS Google Charts via NgMomentum

Here are some demo's of interesting AngularJS widgets using NgMomentum as the controlling script. The chief difference between NgMomentum and AngularJS is the standard ability to load code "as needed", instead of having to load all code at page load. These widgets were re-written to use NgM and MSOS code in order to load them as modules. This allows adding them as needed by user website interaction.
