Tooltips

Each action item supports a tooltip using the standard approach as can be seen above.

ngHide

The speed dial also supports hiding using the standard ng-hide attribute. View the source to see how to apply the animation effect. Hide the speed dial.

ngRepeat

You can easily use ng-repeat with the speed dial, but it requires a slightly different HTML structure in order to support the necessary DOM changes/styling.

Simply ensure that your ng-repeat is on a div (or any other tag) that wraps your items.

Hovering

You can also easily setup the speed dial to open on hover using the ng-mouseenter and ng-mouseleave attributes.

If you want the user to be able to hover over the empty area where the actions will eventually appear, you must also add the md-hover-full class to the speed dial element. Enable "full hover" mode.

Notice that in "full hover" mode, you cannot click on the last "Test" buttons on the toolbar as they are hidden by the speed dial. See the example code and docs for more information.

$mdDialog
