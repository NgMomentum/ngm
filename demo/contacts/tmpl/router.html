NgMomentum UI-Router Example

This section demonstrates the standard integration of AngularJS v1.x UI-Router with NgMomentum. Our NgM version of UI-Router includes mostly changes for enhanced debugging and code clarity. Otherwise, operation is the same.

As you navigate this demo, pay attention to the $state and $stateParams values below.

$state name = {{ diagnostic_state.current.name }}

$stateParams = {{ diagnostic_stateParams }}

$state url = {{ diagnostic_state.$current.url.source }}
